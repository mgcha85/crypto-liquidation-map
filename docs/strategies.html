Methodology

Data Source

1-minute candles from Binance Futures, resampled to target timeframes. Open Interest data at 5-minute intervals.

Features

31 features: 20 liquidation map features (cluster distances, volumes, entropy) + 11 candle features (returns, volatility, ATR).

Labeling

Triple-barrier: +2% profit take, -1% stop loss, 24-hour horizon (scaled by timeframe).

Model

XGBoost binary classifier with class weighting. Parameters: max_depth=4, learning_rate=0.05, n_estimators=150.
